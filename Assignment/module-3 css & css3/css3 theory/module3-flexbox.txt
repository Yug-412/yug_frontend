1. What is CSS Flexbox, and how is it useful for layout design? Explain the terms
flex-container and flex-item.
Ans=CSS Flexbox  is a layout model in CSS that allows you to design complex and responsive web layouts

 Flex-container: is the using parent element hold to flex and using this property display=flex and display: inline-flex
 Flex items: These are the child elements inside a flex container. They are the elements that are arranged in the container according to the flexbox layout properties
